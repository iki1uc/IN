# IN · Mini-Portal (iki1uc System)

IN ist die Eingangsschicht des iki1uc-Systems.

IN = D0  
IN = ROOT  
IN = Startpunkt  
IN = Input-Achse  
IN = IO-Cluster  
IN = SCAN-HUB-Zugang  

Alle Konstellationen = TMP  
Alle Frequenzen = 12e  
Alle Ebenen = 6D  
Alle Sequenzen = 360°

IN ist kompatibel mit:
PIPE, SET, LaTER, LaTERpiFEKT, IO, OI, OUT, SCAN-HUB
